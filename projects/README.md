# projects/

本目录存放用户自己的项目记忆。

## 首次使用

按 `workflows/02-new-project.md` 为你的项目建立记忆目录。每个项目包含：

- `quick-start.md` — 热入口，新对话快速接入
- `00-current-state.md` — 热入口，当前状态和下一步
- `verification/` — 验证记录
- `mistakes/` — 错误复盘
- `decisions/` — 技术决策
- `handoffs/` — 交接记录

## 安全规则

- **禁止保存密钥、密码、Token 或任何认证凭据。**
- **公开仓库不要写入非公开业务资料**（内部接口地址、客户数据、业务逻辑细节等）。
- 本目录不放置示例项目；示例见 `examples/demo-project/`。
- 每个项目独立一个子目录，不跨项目混放文件。
