# Workflow: 新读者上手

## When to use

第一次打开、fork 或评估这个知识库模板时使用。

## Read first

1. `README.md`
2. `AUTHOR.md`
3. `AGENTS.md`
4. `workspace-index.md`
5. `examples/demo-project/quick-start.md`

## Steps

1. 先理解本仓库定位：它是 Agent 工作记忆层，不是业务项目，也不是工具源码。
2. 阅读 `README.md` 的"快速开始"和"目录结构"，确认整体目录分层。
3. 阅读 `AUTHOR.md`，理解为什么要把人类说明和 Agent 规则分开。
4. 阅读 `AGENTS.md`，确认 Agent 新对话的启动链路和硬规则。
5. 阅读 `examples/demo-project/`，看一个项目记忆最小形态长什么样。
6. 决定是否要把本模板复制成自己的知识库；如果要复制，进入 `02-new-project.md`。

## Done when

- 能说清楚热层、温层、冷层分别放什么。
- 知道新对话默认只读 5 个热入口。
- 知道更新文档时不能靠猜，应进入对应 workflow。
